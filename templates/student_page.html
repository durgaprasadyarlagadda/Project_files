Welcome to the student page of the Online Quiz Application. Here, you can take quizzes and view your profile

Profile

In this section you will be able to view your information like username, email address,full name,mobile number and branch.

Quiz

In this section you need to enter a valid exam code to start the quiz.

Quiz Guidelines

You are able to attempt a quiz with unique exam code only one time.

You can attempt multiple quizzes with different exam codes.

After entering the unique exam code the quiz will start and the timer will be set and the quiz will auto submit after the given time.

The quiz is of mcq type and you are able to choose only one option.

You are able to submit the quiz before the given time.
