Welcome to the admin page of the Online Quiz Application. Here, you can manage quizzes, student details, and marks information.

Create Exam

In this section you need to enter a exam code and if the exam code is unique then you will be able to create the quiz.

If the exam code is not unique it will pop a error message.

After successfully entering the unique exam code you need to create the questions which are of mcq type in which only one option is correct.

Marks

Students after succesfully submitting the quiz then their answers will be evaluated and the marks will be redirected to marks section.

The student marks,username,fullname,branch and the respective exam code will be shown in the marks section.

Student Details

The students details will be shown like username,fullname,email,branch and mobile number after the successful registration.
